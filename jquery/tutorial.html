Fun With J-query!

J-Query is an interesting and helpful tool for manipulating web pages we come across on the internet. Below is a demonstration of some of the ways you can modify content in your browser.

First, we need to access the DOM (Document Object Modifier). we can open it by pressing ctrl, shift, and I simultaniously. Once we have access to the window, we can input code to change things. Check out a few examples listed.

Enter the piece of code next to each element into your DOM to see how it changes!

  • Make us Blue! - $(".holder").css("color", "blue");;
  • Change The Background! - $("body").css("background-color", "red");
  • Make me bigger! - $('body').css('font-size','200%')

You can Modify the code to see how it affects your page. whenever youve had your fill of fun, you can enter this last piece of code to make everything dissappear. thanks for checking it out!

$('body').css("display", "none")
